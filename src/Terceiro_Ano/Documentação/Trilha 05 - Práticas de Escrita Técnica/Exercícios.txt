Exercício 1: Escreva uma descrição clara e concisa sobre o processo de instalação
de um software de sua escolha, evitando frases longas e jargões desnecessários.

Para instalar o Google Chrome no Windows, acesse o site oficial google.com/chrome e clique no botão "Download Chrome". Após o download, execute o instalador e siga as instruções exibidas na tela. O processo é rápido e, ao final, o navegador estará pronto para uso. Não é necessário reiniciar o sistema após a instalação.

- Criar um novo projeto utilizando npm init.

	1. Instalar o Node.js

	2. Configurar Variáveis de Ambiente

	Adicione o caminho de instalação do Node.js à variável de ambiente PATH, se 	necessário.
	3. Testar a Instalação

	Abra o terminal e execute um comando para verificar se o Node.js está instalado 	corretamente.
	4. Criar um Novo Projeto

	Inicie um novo projeto interativo e configure as informações básicas, como nome, 	versão e descrição.
