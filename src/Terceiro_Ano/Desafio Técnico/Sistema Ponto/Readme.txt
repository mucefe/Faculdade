=====================
SISTEMA DE REGISTRO DE PONTO
=====================

DESCRIÇÃO:
Aplicação para registro de ponto eletrônico com:
- Tela de login
- Dashboard para funcionários
- Painel administrativo

FUNCIONAMENTO:
1. TELA INICIAL:
   - Login com matrícula e senha
   - Dados de teste:
     * Admin: matrícula "admin", senha "admin123"
     * Funcionário: matrícula "001234", senha "123456"

2. DASHBOARD FUNCIONÁRIO:
   - Registre horários nos 4 botões:
     * Entrada Manhã
     * Saída Manhã
     * Entrada Tarde
     * Saída Tarde
   - Visualize histórico recente

3. PAINEL ADMIN:
   - Cadastro de novos funcionários
   - Relatório de registros do dia
   - Geração de PDF (pdf exemplo incluso)

ARQUIVOS INCLUÍDOS:
/index.html       - Estrutura principal
/index.css        - Estilos personalizados
/index.js         - Lógica do sistema

OBSERVAÇÕES:
1. Este é um protótipo front-end com dados simulados
2. Não possui integração com back-end
3. Funcionalidades completas requerem implementação de API

INSTRUÇÕES DE EXECUÇÃO:
1. Abra o arquivo index.html no navegador
2. Não requer instalação adicional
